Tecnologias

HTML5

É uma linguagem de marcação; é basicamente todo o texto dentro de um site com suas tags, links e semântica.

CSS3

É uma linguagem de estilização; cada cor ou formatação que você está vendo, está declarada no CSS.

JavaScript

Js é uma linguagem de programação; todos os elementos interativos ou dinâmicos são feitos com ele.

Node.Js

Outra linguagem de programação; muito utilizada no back-end, é o que faz a ponte entre o usuário e os dados de um servidor, por exemplo.

Git

Um sistema de controle de versão; imagine um histórico com todas as versões/fases do seu projeto, eis o git.

Sass

Um compilador de estilos; pode-se dizer que é uma extensão do CSS, agregando variáveis, funções e operações aos estilos.
